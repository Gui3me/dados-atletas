# Projeto Atleta

Aplicação em JavaScript capaz de receber informações de um atleta, calcular parâmetros e exibi-los ao usuário.

## Objetivo

Criar uma classe `Atleta` que armazene varios dados do atleta e realize cálculos de categoria, IMC e média válida das suas notas.

## Atributos da Classe

- nome
- idade
- peso
- altura
- notas

## Métodos da Classe Atleta

- calculaCategoria()
- calculaIMC()
- calculaMediaValida()
- obtemNomeAtleta()
- obtemIdadeAtleta()
- obtemPesoAtleta()
- obtemNotasAtleta()
- obtemCategoria()
- obtemIMC()
- obtemMediaValida()
